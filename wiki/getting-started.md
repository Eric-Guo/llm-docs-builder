# Getting Started with llms-txt-ruby

This guide will help you get started with the llms-txt-ruby gem.

## Installation

Add the gem to your Gemfile:

```ruby
gem 'llms-txt-ruby'
```

## Basic Usage

Generate an llms.txt file:

```bash
llms-txt --no-llm
```
